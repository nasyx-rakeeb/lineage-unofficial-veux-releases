## üöÄ About
This is an **Unofficial LineageOS 23** build for the **Redmi Note 11 Pro+ 5G (veux/peux)**, based on the stable Android 16 source code.

This project aims to provide a **Pure, Vanilla** LineageOS experience with **Official-grade standards**, including full OTA support and proper signing.

### ‚ú® Key Features
* **Android 16 (Baklava)**: Based on the stable Google `BP2A.250805.005` release.
* **Native OTA Support**: Receive seamless system updates via **Settings > System > Updater**.
* **Signed Build**: Signed with private **Release Keys** (not test-keys) for better security and Play Integrity.
* **Root Friendly**: Fully supports Magisk's "Install to Inactive Slot" for preserving root across updates.
* **Stable Kernel**: Running on Kernel `5.4.292-LineageOS`.
* **SEPolicy**: Enforcing.
